## はじめに

「ここで決まれば受注、決まらなければ終わり」

クロージングは営業の正念場です。でも実は、**クロージング直前の5分が勝負**だと気づきました。

ChatGPTに3つのことを聞くようにしたら、成約率が体感で1.4倍になりました。

---

## プロンプト①：相手が今一番迷っているポイントを予測させる

```
以下の情報をもとに、この顧客がクロージング直前に最も迷っているポイントをTOP3で予測してください。

・業種：〇〇
・担当者役職：〇〇
・商談での発言：「〇〇が気になる」「△△はどうなの？」
・前回フォローの反応：〇〇

それぞれについて、「どう返すか」の切り返しフレーズも添えてください。
```

**使い方のコツ：** 商談メモを全部渡す。「気になっていた点」を明示すると精度が上がります。

---

## プロンプト②：決断を後押しするメリットの再整理

```
以下の顧客に対して、今日クロージングを成功させるために、
「導入したときの未来」と「導入しなかったときのリスク」を対比形式でまとめてください。

・顧客の課題：〇〇
・自社サービスで解決できること：〇〇
・顧客が気にしている懸念：〇〇
```

**なぜ効くか：** 人は「損失回避」に動きやすい。ChatGPTが整理した「導入しないリスク」の言葉が、自分では気づかなかった切り口を出してくれます。

---

## プロンプト③：クロージングトークの最終確認

```
以下の状況で、顧客に「ご決断をいただく」ためのクロージングトークを3パターン作ってください。

・商談の流れ：〇〇（簡単に）
・顧客の反応：〇〇
・自分のスタイル：ソフト / ミドル / 強め（どれか選ぶ）

それぞれのトークの後に「もし断られたらの切り返し」も添えてください。
```

**3パターン用意する理由：** 当日の空気感で選べるようにするため。準備が多いほど落ち着いて話せます。

## 実際の変化

この3つを使い始めてから：

- クロージング前の「準備時間」が5分で済むようになった
- 「何を言うか」より「どの順番で言うか」が明確になった
- 断られたときの切り返しが事前に頭に入っているので、焦らなくなった

受注できた商談では、「相手が迷っているポイントの予測」がほぼ全部当たっていました。
