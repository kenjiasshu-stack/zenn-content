## はじめに

営業チームで「あの人だけが知っているノウハウ」が属人化していませんか？

トップセールスの成功パターン、よく聞かれる質問への回答、競合との差別化ポイント——こういった情報がチームで共有されれば、全員のレベルが上がります。

ChatGPTを使えば、このナレッジ共有を効率化できます。

---

## 方法1：商談後の気づきをナレッジ化するプロンプト

商談が終わったらすぐにChatGPTに入力します。

```
以下の商談での気づきを、チームで共有できるナレッジ記事にまとめてください。

商談の状況:
- 業種: {業種}
- 商談テーマ: {内容}
- うまくいったこと: {内容}
- うまくいかなかったこと: {内容}
- 相手から出た印象的な質問: {質問内容}

形式:
1. 状況（2〜3行）
2. 学んだこと（箇条書き3つ）
3. 次回使えるフレーズ
4. 注意点
```

これを週1回チームのSlackやNotionに投稿するだけで、チーム全体の財産になります。

---

## 方法2：FAQ自動生成プロンプト

お客様からよく聞かれる質問を収集して、FAQを自動生成します。

```
以下の「お客様からよく聞かれる質問リスト」をもとに、営業チーム用のFAQ集を作成してください。

質問リスト:
- {質問1}
- {質問2}
- {質問3}
（以下続く）

各質問に対して:
1. 回答のポイント（3行以内）
2. 使えるフレーズ
3. 注意点（言ってはいけないこと）

を作成してください。
```

---

## 方法3：トップセールスのトークをテンプレート化

トップセールスの商談を録音・メモして、ChatGPTでパターン化します。

```
以下はトップセールス担当者の商談メモです。
このトークの成功パターンを抽出して、誰でも使えるテンプレートを作成してください。

商談メモ:
{メモ内容}

抽出してほしいもの:
1. アイスブレイクのパターン
2. 課題を引き出す質問の流れ
3. 提案への自然な移行フレーズ
4. クロージングのアプローチ
```

## 実際の効果

このナレッジ共有の仕組みを導入したチームの変化:

| 指標 | 導入前 | 導入後（3ヶ月） |
|------|--------|----------------|
| 新人のアポ率 | 平均8% | 平均14% |
| よくある質問への回答時間 | 5〜10分 | 30秒以内 |
| ロープレの準備時間 | 30分 | 10分 |

チーム全体のベースラインが上がると、売上も自然と底上げされます。

---

## まとめ

ChatGPTで営業ナレッジを共有する3つの方法:

1. **商談後の気づきをナレッジ化** → 属人化を防ぐ
2. **FAQ自動生成** → チーム共通の回答集を作る
3. **トップセールスのトークをテンプレート化** → 成功パターンを横展開

月1回でもこの作業をするだけで、チームの営業力は確実に上がります。
