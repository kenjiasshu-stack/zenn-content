## はじめに

「商談後のフォローが続かない」

これ、営業マンが受注を取り逃がす一番多いパターンです。

最初の商談はうまくいった。でも、その後のフォローが疎かになって、気づいたら競合に取られていた——という経験はありませんか？

ChatGPTを使えば、フォローを「仕組み」にできます。

---

## フォロー管理をChatGPTで仕組み化する方法

### プロンプト1：フォロースケジュールを自動生成する

```
あなたは営業コーチです。以下の商談情報をもとに、最適なフォロースケジュールを作成してください。

商談情報:
- 会社名: ○○株式会社
- 担当者: ○○様
- 商談日: ○月○日
- 検討期間: 約1ヶ月
- 主な懸念点: 導入コスト、社内稟議
- 次のアクション: 見積書の送付

商談日から受注（または失注確定）までの、最適なフォローのタイミングと内容を提案してください。
各フォローで使うメールの件名案も含めてください。
```

---

### プロンプト2：フォローメールを一括で作成する

```
以下の商談情報をもとに、3回分のフォローメールを作成してください。

商談情報:
- 相手: ○○株式会社 ○○様
- 提案内容: ○○サービス
- 主な懸念点: 導入後のサポート体制

【1回目】商談3日後 - 見積書送付のお礼＋資料補足
【2回目】1週間後 - 進捗確認＋懸念点への追加情報提供  
【3回目】2週間後 - 決断を促す最終フォロー

各メールは200字以内、丁寧だが堅すぎないトーンで。
```

---

### プロンプト3：フォロー内容をカスタマイズする

```
先週フォローメールを送ったところ、「まだ社内で検討中」という返信がきました。

この状況に最適な返信メールを作成してください。

状況:
- 前回のフォローから1週間経過
- 相手は前向きだが社内稟議が通らない様子
- 競合他社も検討中の可能性あり

目標: 相手の検討を後押しし、決断を引き出す
```

---

## 実際にやってみた結果

このフォロー仕組み化を始めてから変わったこと：

- **「フォロー忘れ」がゼロになった**（スケジュールが自動で決まるから）
- **メール作成時間が80%削減**（ChatGPTがドラフトを作ってくれる）
- **フォロー品質が均一化**（感覚ではなく戦略的なフォローができる）

以前は「なんとなく1週間後にメールする」だったのが、「商談3日後に○○を送り、1週間後に△△で懸念点を解消する」という計画的なフォローに変わりました。

---

## まとめ

ChatGPTでフォローを仕組み化する3ステップ：

1. **フォロースケジュール自動生成** → タイミングを逃さない
2. **フォローメール一括作成** → 作成時間を大幅削減
3. **状況に応じたカスタマイズ** → 相手の反応に合わせた対応

「フォローが苦手」という方こそ、ChatGPTで仕組み化することをおすすめします。
