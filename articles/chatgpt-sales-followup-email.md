## はじめに

商談が終わったあと、フォローアップメールを書くのに時間がかかっていませんか？

「何を書けばいいか迷う」「毎回似たような文章になる」「送るタイミングを逃す」

ChatGPTを使えば、商談の状況に合わせた最適なフォローアップメールを5分以内に作れます。

---

## 5つのシナリオ別プロンプト

### シナリオ1：好感触・次回商談に繋げたい

```
商談のお礼メールを作成してください。

状況：
- 相手：{会社名} {担当者名}様
- 商談日：{日付}
- 商談の感触：好感触、前向きに検討いただけた
- 次のステップ：来週中に提案書を送付予定

条件：
- 200字以内
- 次回の提案書送付に期待感を持たせる内容
- 押しつけがましくない丁寧なトーン
```

---

### シナリオ2：「検討します」で終わった場合

```
商談後のフォローメールを作成してください。

状況：
- 相手は「検討します」という反応だった
- 具体的な課題は聞き出せたが、決め手に欠けた
- 次回のアクションが決まっていない

目的：
- 相手に「考えてほしいポイント」を一つだけ残す
- 返信しやすい質問を一つ添える
- 売り込み感を出さない

250字以内で作成してください。
```

---

### シナリオ3：競合が入っていることがわかった場合

```
商談後フォローメールを作成してください。

状況：
- 相手が競合他社も検討中だと判明した
- 当社の強みは{強み}
- 競合との差別化ポイントは{ポイント}

目的：
- 比較検討の際に当社を思い出してもらえる内容
- 「一点だけ補足させてください」スタイル

300字以内で作成してください。
```

---

### シナリオ4：資料を送付する際の添え文

```
提案書送付メールの添え文を作成してください。

状況：
- 商談から{X}日後に提案書を送付
- 商談で相手が気にしていた点：{懸念点}
- 提案書で特に見てほしいページ：{ページ内容}

条件：
- 添え文は150字以内
- 「まず{ページ}をご確認ください」という形で誘導する
```

---

### シナリオ5：長期フォロー（1〜3ヶ月後）

```
過去に商談した相手への再アプローチメールを作成してください。

状況：
- 最後の接触から{X}ヶ月経過
- 当時の検討状況：{状況}
- 伝えたい新情報：{新機能・事例・キャンペーンなど}

目的：
- 久しぶりの連絡を自然につなぐ
- アポを取ることが目的だが直接的に求めない

200字以内で作成してください。
```

## まとめ

5つのシナリオ別プロンプトを使い分けることで、どんな商談後でも最適なフォローアップメールが5分で作れます。

商談後のフォローを「面倒な作業」から「勝負の一手」に変えましょう。
