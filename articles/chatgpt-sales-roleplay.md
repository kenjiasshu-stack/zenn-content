# はじめに

営業のロープレって、正直しんどいですよね。

上司の前で「じゃあ始めて」と言われて、緊張して頭が真っ白になる。フィードバックも「トーン落として」「間を作って」みたいな抽象的な話になりがち。

で、ある日ふと思ったんです。**ChatGPTを相手にロープレできないかな**、と。

試してみたら、想像以上に実践的で、本番の商談成功率が明らかに変わった気がします。今日はその方法を紹介します。

---

# ChatGPTロープレの基本セットアップ

まず、こんなプロンプトを使います。

```
あなたは私の商談相手（中小企業の経営者、50代男性）を演じてください。

【状況設定】
- 会社規模: 社員30名の製造業
- 課題: 営業部門の生産性が低く、売上が横ばいが3年続いている
- 性格: 慎重派。新しいものへの投資には慎重だが、結果が出れば動く

私が営業トークを始めたら、リアルな反応を返してください。
「価格が高い」「他社と比較中」などの断り文句も自然に入れてください。

ロープレが終わったら、私の話し方についてフィードバックをください。
```

このセットアップで、本物の商談に近い緊張感が出ます。

---

# 実際にやってみた結果

## やってよかったこと3つ

**1. 断られた時の切り返しを無制限に練習できる**

普通のロープレは1回やるたびに上司の時間をもらう必要があります。でもChatGPT相手なら、同じシナリオを何度でもやり直せる。「さっきの返しはダメだったな」と思ったら即リトライ。

**2. フィードバックが具体的**

ロープレ後に「今の私のトーク、何がよくて何が悪かったか教えて」と聞くと、こんな感じで返ってきます：

> - ヒアリング時間が全体の20%しかなかった（推奨は50%以上）
> - 製品説明が長すぎて相手が置いてきぼりになっていた
> - 「値段が高い」への返しで値下げを匂わせてしまった（価値訴求が弱い）

上司からもらえないレベルの具体性です。

**3. 業界・担当者タイプ別に変えられる**

「今日は警戒心が強いIT企業の担当者役で」「次は話好きな中堅企業の社長役で」と指定するだけで、相手キャラが変わります。幅広い商談パターンに対応できるようになります。

# 応用：事前準備にも使える

商談当日の朝、こんな使い方もします。

```
今日の商談相手は○○株式会社の△△部長です。
業界は○○で、最近××という課題を抱えていると事前情報があります。

この担当者が「うちには必要ない」と言ってきたとき、どう返すべきか
5パターンの返し方を作ってください。
```

これで5分以内に商談の「想定問答集」が完成します。以前は商談1件の準備に30分かけていたのが、今は10分以内に終わります。

---

# まとめ

| 旧来のやり方 | ChatGPT活用後 |
|------|------|
| ロープレは月1〜2回 | 毎日5〜10分できる |
| フィードバックは抽象的 | 具体的で再現性がある |
| 同じ相手タイプしか練習できない | 業界・性格を自由に変更可能 |
| 準備に30分 | 10分以内 |

ChatGPTをロープレ相手にすることで、商談前の不安が減りました。
「この断り文句、昨日も練習した」という感覚は、本番の余裕につながります。

ぜひ一度、商談前に試してみてください。
