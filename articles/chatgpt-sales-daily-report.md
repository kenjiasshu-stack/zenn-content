毎日の営業日報、正直しんどくないですか？

「何を書けばいいかわからない」「同じような内容になる」「上司に怒られる」

私もそうでした。でも、ChatGPTに任せたら日報作成が**10分以下**になりました。

## なぜ営業日報が辛いのか

- 商談の振り返りをしながら文章化する二度手間
- 「課題と改善策」を毎回考えるのが面倒
- 数字の羅列になって「で、何が言いたいの？」と言われる

## ChatGPTに任せるとどう変わるか

入力するのはこれだけ：

```
今日の商談メモを箇条書きで貼り付ける
↓
ChatGPTが日報形式に整形してくれる
```

## 実際に使っているプロンプト

```
あなたは優秀な営業マンの上司です。
以下の商談メモを、営業日報として整形してください。

形式：
【本日の商談件数・訪問先】
【各商談の概要・状況】
【課題と仮説】
【明日のアクション】

商談メモ：
（ここに箇条書きで貼る）
```

## 実際の比較

**Before（以前の日報）：**
> A社訪問。担当者とお話しました。検討中とのこと。フォローします。

**After（ChatGPT使用後）：**
> A社（製造業・従業員200名）の田中部長と打ち合わせ。現行システムの運用コスト削減が急務とのこと。
> 競合X社と比較検討中だが、当社のサポート体制に関心あり。
> 【課題】意思決定者が複数おり、稟議に時間がかかる可能性。
> 【明日のアクション】コスト比較資料を送付し、来週デモの日程調整。

## 上司に「わかりやすくなった」と言われた

この方法を使い始めて1週間で、上司から「最近の日報、読みやすいな」と言われました。

内容が変わったわけじゃない。**構造が変わっただけ**で、評価が上がった。

## 応用：週報・月報にも使える

```
今週の商談記録（添付）をもとに、月次報告資料のサマリーを作ってください。
KPI達成率・主な商談状況・来月の重点施策を含めてください。
```
