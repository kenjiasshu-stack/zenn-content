## はじめに

営業メールを送っても返信が来ない。

その原因の多くは「件名」にあります。どれだけ本文が良くても、件名で開封されなければ読まれません。

ChatGPTを使えば、**開封率が上がる件名のパターン**を5分で複数作れます。

---

## なぜ件名が重要か

BtoB営業メールの平均開封率は20〜25%程度と言われています。つまり4通に3通は件名を見た時点でスルーされています。

開封率を上げる件名の特徴：
- 具体的な数字が入っている
- 相手の課題・業界に関連している
- 短い（30文字以内が理想）
- 「限定感」や「緊急性」がある

---

## ChatGPTで件名を作るプロンプト

### 基本プロンプト

```
営業メールの件名を10パターン作成してください。

【前提情報】
- 送り先：{業種}の{役職}の方
- 提案内容：{何を提案したいか}
- 相手の課題：{相手が抱えていそうな課題}
- 送るタイミング：{初回アプローチ / フォローアップ / 商談後}

以下の型を使って、バリエーションを出してください：
1. 数字を使った型（例：「3分で分かる〇〇」）
2. 課題を直接突く型（例：「〇〇でお困りではないですか？」）
3. 実績・事例を匂わせる型（例：「同業他社が3ヶ月で改善した〇〇」）
4. 好奇心を刺激する型（例：「多くの〇〇担当者が知らない〇〇」）
5. 限定感を出す型（例：「4月限定：〇〇の無料相談」）
```

---

### A/Bテスト用プロンプト

```
以下の件名2つをA/Bテストしたいと思っています。
どちらが開封率が高いと思うか、理由とともに教えてください。
また、両方を上回る可能性がある第3の案も提案してください。

件名A：{件名A}
件名B：{件名B}

業種：{業種}
役職：{役職}
```

---

### フォローアップメール用プロンプト

```
以下の状況でフォローアップメールを送ります。
前回と差別化した件名を5つ作ってください。

前回の件名：{前回送った件名}
前回の結果：{開封されたが返信なし / 開封されなかった}
今回追加できる情報：{新しい事例 / 期間限定オファー / 相手の近況}
```

---

## 実際に使ってみた結果

以前の件名：「〇〇のご提案について」
→ 開封率：18%

ChatGPTで作り直した件名例：「【同業他社3社が導入済み】〇〇を3ヶ月で改善した方法」
→ 開封率：31%（約1.7倍）

件名一つで開封率が大きく変わります。

---

## 件名で避けるべきNGワード

ChatGPTに聞くとさらに詳しく教えてくれますが、基本的なNGワード：

- 「ご確認ください」（曖昧すぎる）
- 「重要なお知らせ」（スパム判定されやすい）
- 「無料」（迷惑メールフォルダ行きリスク）
- 「お世話になっております」（件名に入れるもの）

```
営業メールの件名として避けるべきNGワードと、その理由を教えてください。
また、NGワードを使いたくなる状況の代替表現も合わせて教えてください。
```

---

## まとめ

ChatGPTで件名を最適化する手順：

1. 業種・役職・提案内容・課題を整理する
2. 5つの型で10パターン生成する
3. A/Bテスト用プロンプトで絞り込む
4. 実際に送ってデータを取る
5. 結果をもとに次回のプロンプトを改善する

件名の改善は即効性があります。今使っている件名をChatGPTに見せて、改善案を出してもらうところから始めてみてください。
