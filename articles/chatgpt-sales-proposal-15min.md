# はじめに

「提案書を作るのが時間がかかって困っている」

営業職をやっていると、こう感じることが多いと思います。私もそうでした。

商談の翌日に提案書を仕上げようとすると、気づいたら2〜3時間経っていた。でもChatGPTを使い始めてから、その作業が**15〜20分**で終わるようになりました。

この記事では、実際に私が使っているプロンプトと手順を公開します。

---

## 従来の提案書作成の流れ

以前は、提案書を作るとき、こんな流れでやっていました。

1. 商談メモを見返す（10分）
2. 何を書くか考える（20分）
3. 構成を決める（15分）
4. 文章を書く（40分）
5. 体裁を整える（15分）

合計：**約100分**

これが毎週3〜4件あったので、週に5〜6時間が提案書作成に消えていました。

---

## ChatGPTを使った新しい流れ

ChatGPTを使うようになってからは、こうなりました。

1. 商談メモをChatGPTに貼る（2分）
2. 構成案を出力させる（1分）
3. 各セクションの文章を生成させる（5分）
4. 微調整して体裁を整える（7分）

合計：**約15分**

---

## 実際に使っているプロンプト

### Step 1: 商談メモから構成を作る

```
以下の商談メモをもとに、提案書の構成案を作成してください。

【商談メモ】
[商談内容を箇条書きでここに貼る]

出力形式：
- 提案書タイトル
- 各章の見出し（5〜7項目）
- 各章に入れるべきポイント（2〜3点）
```

### Step 2: 課題と提案の核心を作る

```
以下の情報をもとに、提案書の「課題認識」と「解決策」のセクションを書いてください。

顧客の現状課題：[課題を3点]
自社サービス：[サービス名・特徴]
顧客が得られる効果：[期待する効果]

文体：ビジネス向け、簡潔に、専門用語は避けて平易な言葉で
```

### Step 3: 導入効果をリアルに見せる

```
以下の情報から「導入効果」セクションの文章を作成してください。

サービス名：[サービス名]
主な効果：
- [効果1]
- [効果2]
- [効果3]

具体的な数字や「導入前/導入後の比較」を使って、読む人がイメージしやすい形にしてください。
```

### Step 4: まとめと次のステップ

```
以下の状況を踏まえて、提案書の「まとめとご提案」セクションを書いてください。

現在の状況：[顧客の状態、例：前向きだが社内承認が必要]
提案している内容：[サービス名・プラン]
次のアクション：[例：来週のデモ実施・見積提出など]

購入を強制するのではなく、自然に次のステップへ進めるような書き方にしてください。
```

## やってみて気づいたこと

### 良かった点

**思考の整理が早くなる**

ChatGPTに打ち込む過程で、「何を伝えたいのか」が自分でも整理されます。「商談メモを貼るだけ」なのに、構成が出てきたときに「そうそう、これが言いたかった」と思えることがよくあります。

**均質な品質が保てる**

疲れている日でも、忙しい日でも、一定品質の提案書が出せるようになりました。「今日は頭が働かないから文章がイマイチ」がなくなります。

### 注意点

**そのままでは使えない**

ChatGPTの出力をそのまま提案書にするのはNGです。顧客固有の情報（社名・担当者名・具体的な数字）は自分で確認・入力が必要です。

あくまでも「叩き台を高速で作るツール」として使うのがベストです。

**社内情報をそのまま入力しない**

顧客名や機密情報をそのまま入力するのは避けましょう。「A社」「B社長」のように置き換えて使う習慣をつけると安心です。

## まとめ

ChatGPTを使った提案書作成で変わったこと：

- 作成時間：100分 → 15分
- 1週間の節約時間：約6〜8時間
- 提案書の品質：安定（疲労状態でも均質）

この時間を商談準備や顧客対応に使えるようになったのが、一番の収穫でした。

最初は「AIが作った文章は薄い」と思っていましたが、自分でプロンプトを工夫すれば十分実用的なレベルになります。ぜひ試してみてください。
