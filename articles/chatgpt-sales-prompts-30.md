# はじめに

営業職の方が**今すぐ実務で使えるChatGPTプロンプト30選**をまとめました。

「AIを使ってみたいけど、何を入力すればいいかわからない」という方向けに、コピペでそのまま使えるプロンプトを厳選しました。

---

## 商談準備編

### 1. 企業リサーチ＆課題仮説

```
以下の企業の情報を調べ、商談に向けた課題・ニーズの仮説を3つ作成してください。
企業名：[企業名]
業種：[業種]
自社商品/サービス：[自社の商品・サービス名]
```

### 2. 提案書の構成案

```
以下の条件で提案書の構成案を作成してください。
対象顧客：[顧客名・業種]
課題：[顧客の課題]
提案内容：[自社サービス名]
構成は「課題認識 → 解決策 → 導入効果 → 事例 → 費用・スケジュール」の順で。
```

### 3. アイスブレイク話題

```
初回営業訪問のアイスブレイクとして使える話題を3つ提案してください。
訪問先：[業種・企業規模]
訪問日：[日付・季節]
共通の話題になりやすいビジネストレンドも含めてください。
```

### 4. ヒアリングシート設計

```
以下の商品を販売するための初回ヒアリングシートを作成してください。
商品：[商品名]
業種：[ターゲット業種]
確認したい項目：予算・導入時期・決裁者・現状課題・競合検討状況
```

### 5. 競合比較トーク

```
顧客から「競合の○○社と何が違うの？」と聞かれました。
以下の情報をもとに、わかりやすい差別化トークを作成してください。
自社商品の強み：[強み]
競合商品：[競合名]
顧客の課題：[課題]
```

---

## 商談中編

### 6. 顧客ニーズの言語化（SPIN分析）

```
以下の顧客の発言から、潜在的なニーズと本質的な課題を分析してください。
顧客の発言：「[顧客の発言]」
業種：[業種]
SPIN分析（状況・問題・示唆・解決）で整理してください。
```

### 7. 事例紹介トーク

```
以下の導入事例をもとに、商談で使える事例紹介トークを作成してください。
事例の概要：[業種・課題・導入後の効果]
聞き手：[業種・役職]
話す時間：1〜2分
具体的な数字を使い、聞き手が「自分ごと」と感じやすい内容に。
```

### 8. クロージングトーク

```
以下の状況でクロージングに向けた一押しのトークを作成してください。
顧客の状態：「前向きだが決断できていない」
主な懸念：[費用 / 社内承認 / 他社との比較など]
自社商品の最大の強み：[強み]
自然で誠実な言い方にしてください。
```

### 9. 価格交渉への対応

```
顧客から「もう少し安くならない？」と言われました。
値引きせずに納得してもらえる対応トークを作成してください。
現在の価格：[金額]
価格の根拠：[品質・サポート・導入実績など]
```

### 10. 断り文句への切り返し

```
営業の商談で以下の断り文句を言われました。
自然で押しつけがましくない切り返しトークを3パターン作成してください。
断り文句：「今は予算がありません」
自社商品：[商品名・特徴]
```

## 商談後・フォローアップ編

### 11. 議事録→アクション整理

```
以下の商談メモから、ネクストアクションと確認事項を整理してください。

[商談メモをここに貼り付け]

出力形式：
- 自社のアクション
- 顧客への確認事項
- 次回商談の提案テーマ
```

### 12. フォローメール作成

```
商談後のフォローメールを作成してください。
商談内容のポイント：[3点以内で箇条書き]
次のステップ：[例：資料送付・デモ日程調整など]
文体：ビジネス敬語、簡潔に3段落以内
```

### 13. 社内稟議文テンプレート

```
顧客の担当者が社内で稟議を通すための稟議文のテンプレートを作成してください。
導入サービス：[サービス名]
導入目的：[課題解決・コスト削減など]
費用：[金額]
期待効果：[効果]
担当者が上司に説明しやすい構成に。
```

### 14. 失注後の関係継続メール

```
今回は受注に至らなかった顧客への関係継続メールを作成してください。
失注理由：[例：他社に決定・予算不足・時期尚早]
今後の可能性：[例：次期検討時・追加ニーズ発生時]
印象を良く保ちながら、次の接点につながる内容に。
```

## テレアポ編

### 15. テレアポスクリプト作成

```
電話営業用のトークスクリプトを作成してください。
商品：[商品名]
ターゲット：[業種・役職]
目的：アポイント獲得
想定される断り文句への切り返しも含める。
全体を2分以内で話せる分量に。
```

### 16. 担当者変更時のアポ取り

```
前任担当者が退職し、新担当者に初めて電話する際のトークを作成してください。
前任との関係：[良好 / 案件進行中 / 提案中]
自社商品：[商品名]
新担当者の役職：[役職]
```

### 17. リスト精査のための事前リサーチ

```
以下の企業にテレアポする前に確認すべき情報を整理してください。
企業名：[企業名]
業種：[業種]
自社商品：[商品名]
確認項目：決裁者の役職・アポが取れやすい時間帯・競合利用状況
```

## 提案書・資料作成編

### 18. 1枚サマリー（エグゼクティブ向け）

```
以下の提案内容を、決裁者向けの1枚サマリーに変換してください。
提案内容：[詳細な提案文]
決裁者の関心事：[コスト削減 / 売上拡大 / リスク回避]
箇条書き5点以内でまとめてください。
```

### 19. ROI計算シートの説明文

```
以下のROIデータをもとに、顧客が納得しやすい説明文を作成してください。
現状コスト：[金額/月]
導入コスト：[金額]
期待される削減効果：[金額/月]
回収期間の目安も含めてください。
```

### 20. Q&A想定問答集

```
以下の提案に対して顧客から想定される質問と回答を10個作成してください。
提案内容：[提案の概要]
顧客の懸念ポイント：[費用 / 効果 / 導入工数 / サポート体制]
```

## マネジメント・チーム活用編

### 21. 営業会議のアジェンダ作成

```
以下の目的に合わせた営業会議のアジェンダを作成してください。
目的：[月次レビュー / 案件共有 / 戦略策定など]
参加者：[人数・役職]
会議時間：60分
重要度の高い議題から優先して配置してください。
```

### 22. 営業メンバーへのフィードバック文

```
以下の状況に合わせて、営業メンバーへの建設的なフィードバック文を作成してください。
メンバーの状況：[案件が失注続き / 目標未達 / 新人で伸び悩みなど]
良い点：[具体的に]
改善点：[具体的に]
前向きに行動できるトーンにしてください。
```

### 23. ロープレシナリオの作成

```
以下の条件でロールプレイング用のシナリオを作成してください。
商品：[商品名]
想定顧客：[業種・規模・役職]
難易度：[初級 / 中級 / 上級]
断り文句や厳しい質問も含めてください。
```

## メール・文書作成編

### 24. 謝罪メール（クレーム対応）

```
以下の状況に合わせたクレーム対応メールを作成してください。
クレームの内容：[具体的な内容]
こちらの過失度：[全面的 / 一部]
今後の対応策：[具体的に]
誠実かつ具体的な対応を示す内容にしてください。
```

### 25. 値上げ通知メール

```
取引先への値上げ通知メールを作成してください。
値上げ幅：[%または金額]
値上げ理由：[原材料費・人件費・物流コストなど]
適用時期：[日付]
関係を損なわず、理解を得やすいトーンで。
```

### 26. 新サービス案内メール

```
既存顧客への新サービス案内メールを作成してください。
新サービス名：[サービス名]
既存顧客との関係：[導入済みサービス名]
新サービスのベネフィット：[3点]
既存サービスとの相乗効果も伝えてください。
```

## 分析・戦略立案編

### 27. 競合他社のSWOT分析

```
以下の競合他社のSWOT分析を行ってください。
競合名：[企業名]
業種：[業種]
公開されている情報をもとに、自社との比較も含めて分析してください。
```

### 28. ターゲット顧客のペルソナ作成

```
以下の条件でターゲット顧客のペルソナを作成してください。
商品：[商品名]
想定業種：[業種]
役職：[例：営業部長・経営者]
課題・悩み・情報収集方法・購買決定プロセスを含めてください。
```

### 29. 今月の戦略を1枚で整理

```
以下の状況をもとに、今月の営業戦略を1枚にまとめてください。
目標：[売上目標・件数]
現状：[達成率・パイプライン状況]
課題：[最大の障壁]
重点アクション3つと優先度を明記してください。
```

### 30. 失注分析レポート

```
以下の失注案件を分析し、今後の改善点をレポートにまとめてください。
案件数：[件数]
主な失注理由：[価格 / 競合負け / ニーズミスマッチなど]
担当者別の傾向も含めて、再現性のある改善策を提案してください。
```
