# はじめに

営業で一番キツいのは、断られたときの切り返しです。

「今は予算がなくて」「担当者に聞いてみないと」「他社と比較中で」

こういった断り文句への返し方って、経験で覚えるものだと思っていました。でも、ChatGPTに考えさせてみたら、**予想以上に使えるトークが出てきた**んです。

この記事では、実際に試した方法とプロンプトを公開します。

---

## 試したきっかけ

テレアポの成果が安定しなくて悩んでいた時期がありました。

断られること自体は仕方ない。でも、断られた後に何も言えずに終わる商談が多くて。上司に相談しても「場数を踏め」で終わる。先輩のトークを聞いても、自分のスタイルに合わない。

そこで「ChatGPTに切り返しトークを考えさせたら使えるのか？」と試してみました。

---

## 使ったプロンプト

```
営業の電話（テレアポ）で以下の断り文句を言われました。
自然で押しつけがましくない切り返しトークを3パターン作成してください。

断り文句：「今は予算がありません」
自社商品：[商品名・特徴]
ターゲット：[業種・規模]

条件：
- 否定しない（「でも〜」「しかし〜」で返さない）
- 共感から入る
- 次の接点につながるように締める
- 1パターン5〜6文程度
```

---

## 実際に出てきた切り返し（「予算がない」パターン）

ChatGPTが出してきた3パターンを紹介します。

### パターンA：共感 + 来期への布石

> そうですね、今の時期はどこの会社さんも予算が厳しいタイミングですよね。
> 無理にご提案するつもりはないのですが、来期の計画を立てる際のご参考になればと思いまして。
> もし来期の予算取りをご検討の際に、一度お話しだけでも聞いていただけますか？

### パターンB：費用対効果の提示

> おっしゃる通りです。ただ、導入いただいているお客様から「初月で費用回収できた」という声もいただいていまして。
> 予算の問題というより、投資対効果の観点から見ていただけると、少し違うご判断になるかもしれません。
> 5分だけ数字をご説明させていただいてもよいでしょうか？

### パターンC：タイミングの把握

> そうですか、承知いたしました。無理にご提案するつもりはありません。
> 参考までに教えていただけますか？来期の予算計画は、いつ頃から始まりますか？
> もしよろしければ、その時期に改めてご連絡させていただければと思います。

## 実際に使ってみた感想

### 使えた点

**3パターン出してくれるので選べる**

自分のトーンに合うものを選べるのが良かったです。私はパターンCが一番自然に話せた。

**フレームワークが整理される**

「共感→提案→次の接点」の流れが毎回ちゃんと組み立てられているので、頭の中が整理される感覚がありました。

**断り文句ごとにカスタマイズできる**

「今は忙しい」「社内で検討します」「担当者が変わった」など、よくある断り文句をまとめてChatGPTに流せます。自分だけの切り返し集が作れる。

### 注意点

**そのまま読むと不自然**

ChatGPTの文章は、そのまま電話で読むと違和感があることも。自分の言葉に翻訳する作業が必要です。

**相手の反応には対応できない**

あくまで「こう言われたら、こう返す」という定型のパターンです。相手が予想外の返し方をした場合は、自分で考えるしかない。

## 断り文句別のプロンプトテンプレート

よく来る断り文句をパターン化してまとめておくと便利です。

```
断り文句：「担当者に聞いてみます」
→ 担当者につないでもらうためのプッシュトーク
```

```
断り文句：「今は検討中です」
→ 他社との比較に負けないための差別化トーク
```

```
断り文句：「必要になったら連絡します」
→ 連絡を待つのではなく、タイムラインを確認するトーク
```

これをChatGPTに流して、自分だけの「断り文句対応集」を作っておくと、テレアポ前の準備が一気に楽になります。

## まとめ

ChatGPTで切り返しトークを作って気づいたこと：

- 完璧なトークはすぐ出ない（3パターン出して、1つ選ぶ感覚）
- 経験者に聞いた話より、体系的に整理されていることもある
- 使えるかどうかは「自分の言葉に直せるか」で決まる

「AIが営業トークを考えるなんて」と最初は疑っていましたが、実際に試してみると、「たたき台を高速で作るツール」として十分機能します。

ぜひ自分の断り文句リストを作って、試してみてください。
